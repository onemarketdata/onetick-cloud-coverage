# Deutsche Boerse / STOXX Indices Enumerations

Deutsche Boerse operates the Frankfurt Stock Exchange and provides indices like the STOXX indices covering European and global markets across asset classes.

The following fields have Enumerations:

* COND - Condition code
  : ‘””’,”No condition”
    “A”,”Official (all constituents open)”
    “I”,”Indicative (outside trading hours, or too few constituents open for normal index calculation)”
    “M”,”End of month value”
    “R”,”Representative (not all constituents open, but enough for the index to be considered representative)”
    “U”,”Unverified (value deviates from expected parameters, needs to be checked by the exchange)”
